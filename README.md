# text-clamp

## Installation

You can use this package on the server side as well as the client side.

### [Node.js](http://nodejs.org/):

~~~
npm install text-clamp
~~~

## API

~~~ javascript
clamp(element, options)
~~~

## Usage

~~~ javascript
import {clamp} from 'text-clamp'
clamp(element, {
    clamp:2
})

~~~
